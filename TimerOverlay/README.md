# 替身计时器（Windows 悬浮窗版）

不抢游戏焦点的全局热键计时器，专为 MuMu 模拟器横版格斗游戏设计。

## 工作原理

热键监听在 **Windows 系统层**，不经过 MuMu 的键位映射，所以无论你在游戏里同时按多少个键（WASD移动、技能、鼠标拖视角），按下触发键都能立即启动倒计时。悬浮窗置顶且**不抢焦点**，显示在游戏上方但不影响游戏操作。

## 编译（GitHub Actions，无需装任何软件）

1. 上传到 GitHub 仓库
2. Actions 自动编译
3. 下载 `替身计时器.exe`

## 使用

1. 双击 `替身计时器.exe` 运行
2. 屏幕左上角出现红色圆圈，显示数字
3. **默认按 F8 键**开始倒计时，结束后自动回到待机数字
4. 鼠标拖动圆圈可移动位置

## 自定义配置

首次运行会在 exe 同目录生成 `config.json`，用记事本打开即可修改：

```json
{
    "hotkey": "F8",        // 触发键：F1-F12、A-Z、0-9、NUM0-NUM9、SPACE、TAB 等
    "countdown": 14,       // 倒计时秒数
    "window_x": 100,       // 悬浮窗 X 坐标
    "window_y": 100,       // 悬浮窗 Y 坐标
    "circle_size": 80,     // 圆圈大小
    "font_size": 34        // 数字字体大小
}
```

改完保存，重新运行 exe 生效。

### 支持的触发键

- 功能键：F1 ~ F12
- 字母：A ~ Z
- 数字：0 ~ 9
- 小键盘：NUM0 ~ NUM9
- 其他：SPACE、TAB、INSERT、HOME、END、PAGEUP、PAGEDOWN

建议用 **F8、F9** 等功能键，或**小键盘数字键**，避开游戏常用的 WASD 和技能键。

## 颜色提示

- 🔴 红色（>10秒）
- 🟠 橙色（6-10秒）
- 🟡 黄色（1-5秒）
- 倒计时结束自动恢复红色待机数字
